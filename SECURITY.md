Reporting a Vulnerability

If you discover a security vulnerability in the Fitcoin (FTC) ecosystem, please report it responsibly.

How to Report

You can report vulnerabilities through the following channels:

Email: security@futurecoin.in
Website: https://www.futurecoin.in
Community: https://www.blackmercury.in

Please include the following information in your report:

Description of the vulnerability

Steps to reproduce the issue

Affected components (smart contract, wallet, mining system, API, etc.)

Potential impact

Suggested fix (if available)



---

Response Time

Our security team aims to respond according to the following timeline:

Acknowledgement: within 48 hours

Initial Investigation: 3–5 days

Fix / Mitigation Release: depending on severity



---

Responsible Disclosure

We request security researchers to:

Avoid public disclosure before the issue is fixed

Provide sufficient time for patches to be released

Not exploit vulnerabilities for personal gain


Researchers helping improve Fitcoin security may be recognized by the community and potentially rewarded in Fitcoin (FTC).


---

Security Scope

Security reports may include vulnerabilities related to:

Smart contracts

Mining reward algorithms

Wallet systems

API services

Blockchain integration

Authentication systems

Economic reward logic (Calorie Burn → FTC Reward)
